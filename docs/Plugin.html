BusEngine.dll

BusEngine.Plugin (Abstract Class)

Класс отвечающий за работу плагинов для BusEngine. Плагины подключаются в файле настроек проекта "*.busproject" или настрок редактора "busengine.busengine".

Методы

(void)BusEngine.Plugin.Initialize() Метод (виртуальный) уровня класса запускает плагин во время выполнения "BusEngine.Engine.Initialize()".
(void)BusEngine.Plugin.Initialize(string) Метод (виртуальный) уровня класса запускает плагин во время выполнения "BusEngine.Engine.Initialize()" столько раз, сколько плагинов подключено. Первый параметр string строка выводит название файла плагина который сейчас был загружен.
(void)BusEngine.Plugin.InitializeCanvas() Метод (виртуальный) уровня класса запускает плагин во время выполнения "BusEngine.UI.Canvas.Initialize()".
(void)BusEngine.Plugin.Shutdown() Метод (виртуальный) уровня класса останавливает плагин для уничтожения объектов плагина и освобождения оперативной памяти при остановки BusEngine.
(void)BusEngine.Plugin.OnLevelLoading(string) Метод (виртуальный) уровня класса запускает плагин во время начала загрузки игрового уровня "BusEngine.Level". Первый параметр string строка выводит название игрового уровня который сейчас начал загружаться.
(void)BusEngine.Plugin.OnLevelLoaded(string) Метод (виртуальный) уровня класса запускает плагин во время завершения загрузки игрового уровня "BusEngine.Level". Первый параметр string строка выводит название игрового уровня который сейчас окончил загружаться.
(void)BusEngine.Plugin.OnGameStart() Метод (виртуальный) уровня класса запускает плагин во время начала управления игровым персонажем или начала отсчёта времени игры.
(void)BusEngine.Plugin.OnGameStop() Метод (виртуальный) уровня класса запускает плагин во время окончании управления игровым персонажем или окончании отсчёта времени игры.
(void)BusEngine.Plugin.OnGameUpdate() Метод (виртуальный) уровня класса запускает плагин во время отрисовки каждого кадра.
(void)BusEngine.Plugin.OnClientConnectionReceived(int) Метод (виртуальный) уровня класса запускает плагин во время начала подключения к игровому серверу. Первый параметр int строка выводит числовой id игрока который начал подключаться к серверу.
(void)BusEngine.Plugin.OnClientReadyForGameplay(int) Метод (виртуальный) уровня класса останавливает плагин во время окончания подключения к игровому серверу. Первый параметр int строка выводит числовой id игрока который окончил подключаться к серверу.
(void)BusEngine.Plugin.OnClientDisconnected(int) Метод (виртуальный) уровня класса останавливает плагин во время окончания игры в игровом сервере. Первый параметр int строка выводит числовой id игрока который начал отключаться от сервера.

  62.             BusEngine.Log.Info("MyPlugin OnClientDisconnected: {0}", channelId);

  64.     }
