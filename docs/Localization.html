BusEngine.Localization (Class)

Класс отвечающий за работу с Языковыми файлами. Это мультиязычность движка. В данной версии BusEngine включена поддержка Белорусского, Русского и Украинского языка.

Методы

var _localization = (BusEngine.Localization)new BusEngine.Localization() Метод конструктор (нестатический) уровня класса не принимает никаких параметров.
(void)_localization.Load(string) Метод (нестатический) уровня класса принимает string строку название языка для загрузки языка.
(string)_localization.GetLanguage(string) Метод (нестатический) уровня класса принимает string строку для получения из языкового файла значение указанной переменной. "string text_start_game = _localization.Get("text_start_game");" Если переменной нет в файле, то вернёт название переменной.
(string)_localization.GetLanguage(string, string) Метод (нестатический) уровня класса принимает string строку в первый параметр для указания названия языкового файла и string строку во второй параметр для получения из языкового файла значение указанной переменной. "string text_start_game = _localization.Get("myplugin", "text_start_game");" Если переменной нет в файле или файла, то вернёт название переменной.
(void)_localization.SetLanguage(string, string) Метод (нестатический) уровня класса принимает string строку для установки в языковой файл значение указанной переменной. "_localization.Set("text_start_game", "Начать игру");".
(void)_localization.SetLanguage(string, string, string) Метод (нестатический) уровня класса принимает string строку для установки в языковой файл значение указанной переменной. "_localization.Set("text_start_game", "Начать игру");".
(string)BusEngine.Localization.GetLanguageStatic(string) Метод (статический) уровня класса принимает string строку для получения из языкового файла значение указанной переменной. "string text_start_game = BusEngine.Localization.GetLanguage("text_start_game");" Если переменной нет в файле, то вернёт название переменной. Не реакомендуется использовать, если хотите получить переменную не из языкового файла по умолчанию.
(string)BusEngine.Localization.GetLanguageStatic(string, string) Метод (статический) уровня класса принимает string строку в первый параметр для указания названия языкового файла и string строку во второй параметр для получения из языкового файла значение указанной переменной. "string text_start_game = BusEngine.Localization.GetLanguage("myplugin", "text_start_game");" Если переменной нет в файле, то вернёт название переменной. Реакомендуется использовать этот метод для быстрого доступа к переменной и когда не создавали объект "BusEngine.Localization".
(void)BusEngine.Localization.SetLanguageStatic(string, string) Метод (статический) уровня класса принимает string строку для установки в языковой файл значение указанной переменной. "BusEngine.Localization.SetLanguage("text_start_game", "Начать игру");".
(void)BusEngine.Localization.SetLanguageStatic(string, string) Метод (статический) уровня класса принимает string строку для установки в языковой файл значение указанной переменной. "BusEngine.Localization.SetLanguage("text_start_game", "Начать игру");".

Поля

(string)_localization.LanguageDefault Поле (нестатическое) уровня класса принимает string строку для установки по умолчанию названия файла языковых перемен который будет загружаться, если не найден файл указанный в "_localization.Language". По умолчанию "Belarusian".
(string)_localization.Language Поле (нестатическое) уровня класса принимает string строку для установки названия файла языковых перемен, который нужно загрузить сейчас. По умолчанию не установлено, загружается "BusEngine.Localization.LanguageDefault".
(string)_localization.File Поле (нестатическое) уровня класса принимает string строку для установки иного названия файла языковых перемен. По умолчанию не установлено, загружается "BusEngine.Localization.Language". Если указать своё название, то этот файл должен находиться в папке с названием языка указанным в "_localization.Language".
(string)_localization.Format Поле (нестатическое) уровня класса принимает string строку для установки расширения файла языковых перемен. По умолчанию "cfg".

События

(void)_localization.OnLoad Событие (нестатическое) уровня класса принимает метод с двумя параметрами (object, string). Событие "BusEngine.Browser.PostMessage" будет вызывать ваш метод, когда вы вызываете загрузку языка "_localization.Load(string)" и передавать объект "BusEngine.Localization()" в первый параметр, и строку содержащую имя языка во второй параметр вашего метода.
(void)BusEngine.Localization.OnLoadStatic Событие (статическое) уровня класса принимает метод с двумя параметрами (object, string). Событие "BusEngine.Browser.PostMessage" будет вызывать ваш метод, когда вы вызываете загрузку языка "_localization.Load(string)" и передавать объект "BusEngine.Localization()" в первый параметр, и строку содержащую имя языка во второй параметр вашего метода.

Пример файла Program.cs

В коде отражён запуск перевода при старте своего плагина с названием MyPlugin.

  1. /** API BusEngine.Game - пользовательский код */
  2. namespace BusEngine.Game {
  3.     /** API BusEngine.Plugin */
  4.     public class MyPlugin : BusEngine.Plugin {
  5.         // при запуске BusEngine до создания формы
  6.         public override void Initialize() {
  7.             BusEngine.Log.Info("1 {0}", BusEngine.Localization.GetLanguage("error"));
  8.             BusEngine.Localization language = new BusEngine.Localization();
  9.             language.LanguageDefault = "Belarusian";
  10.             language.Language = "Ukrainian";
  11.             language.File = "";
  12.             language.Format = "cfg";
  13.             language.Initialize();
  14.             // или
  15.             //new BusEngine.Localization().Load("Ukrainian");
  16.             BusEngine.Log.Info("2 {0}", BusEngine.Localization.GetLanguage("error"));
  17.             BusEngine.Localization.SetLanguage("error", BusEngine.Localization.GetLanguage("error_warning"));
  18.             BusEngine.Log.Info("3 {0}", BusEngine.Localization.GetLanguage("error"));
  19.         }
  20.     }
  21.     /** API BusEngine.Plugin */
  22. }
  23. /** API BusEngine.Game - пользовательский код */
