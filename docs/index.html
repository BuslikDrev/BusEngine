API BusEngine

BusEngine.Audio.Play(string) Метод уровня класса для запуска Аудио файла.
BusEngine.Browser.Start(string) Метод уровня класса принимает string строку для запуска браузера и загрузки указанной страницы.
BusEngine.Browser.Shutdown() Метод уровня класса для закрытия "BusEngine.Browser".
BusEngine.Engine.GenerateStatLink() Метод уровня класса для генерации перемен уровня класса таких, как "BusEngine.Engine.DataDirectory", "BusEngine.Engine.Platform" и других необходимых действий для работы API BusEngine.
BusEngine.Engine.DataDirectory Переменная уровня класса выводит string строку полного пути до папки файлов проекта.
BusEngine.Engine.Platform Переменная уровня класса выводит string строку текущей платформы.
BusEngine.Engine.Shutdown() Метод уровня класса для закрытия "BusEngine".
BusEngine.Localization.LanguageDefault Переменная уровня класса принимает string строку для установки по умолчанию названия файла языковых перемен который будет загружаться, если не найден файл указаный в "BusEngine.Localization.Language". По умолчанию "Belarusian".
BusEngine.Localization.Language Переменная уровня класса принимает string строку для установки названия файла языковых перемен. По умолчанию не установлено, загружается "BusEngine.Localization.LanguageDefault".
BusEngine.Localization.File Переменная уровня класса принимает string строку для установки иного названия файла языковых перемен. По умолчанию не установлено, загружается "BusEngine.Localization.Language". Если указать своё название, то этот файл должен находиться в папке с названием языка указанным в "BusEngine.Localization.Language".
BusEngine.Localization.Format Переменная уровня класса принимает string строку для установки расширения файла языковых перемен. По умолчанию "cfg".
BusEngine.Localization.GetLanguage(string) Метод уровня класса принимает string строку для получения из языкового файла значение указанной переменной. "string text_start_game = BusEngine.Localization.GetLanguage("text_start_game");" Если переменной нет в файле, то вернёт название переменной.
BusEngine.Localization.SetLanguage(string, string) Метод уровня класса принимает string строку для установки в языковой файл значение указанной переменной. "BusEngine.Localization.SetLanguage("text_start_game", "Начать игру");".
